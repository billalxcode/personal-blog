# Apa itu Pandas?

Pandas adalah sebuah pustaka perangkat lunak yang ditulis untuk bahasa pemrogramman Python, fungsi dari Pandas ini yaitu untuk melakukan manipulasi dan analisa data. Secara khusus, Pandas menawarkan struktur data dan operasi yang dapat memanipulasi tabel numerik (numeric) dan deret waktu (time series). Pandas ini rilis di bawah lisensi [BSD](https://en.wikipedia.org/wiki/3-clause_BSD_license).

# Sejarah Singkat

[Wes McKinney](https://en.wikipedia.org/wiki/Wes_McKinney) orang yang pertama kali membangun Pandas ini pada tahun 2008. Saat itu ia masih berada di [AQR Capital Management](https://en.wikipedia.org/wiki/AQR_Capital), ia mengembangkan Pandas ini karena kebutuhan akan alat berkinerja tinggi dan fleksibel untuk melakukan analisis kuantitatif pada data keuangan.

# Model Data

Pandas memiliki struktur data yang disebut `Series` dan `DataFrame`. Data untuk koleksi dapat diimport dari berbagai format seperti `CSV, JSON, Parquet, SQL, Microsoft Excel`.

## Apa itu Data Series?

Jika Anda sudah mempelajari konsep array, atau suatu nilai yang memiliki satu dimensi, maka Series itu sebuah jenis data yang mirip dengan konsep yang disebutkan tadi. Objek ini dapat menyimpan data dari berbagai tipe data (integer, float, string, dll). Series ini hanya memiliki satu kolom, yang dimana jenis ini merupakan struktur data fundamental dalam Pandas dan umum digunakan untuk manipulasi dan analisis data. Series dapat dibuat dari list, array, dictionary dan objek Series yang sudah ada.

Mari kita contohkan kode series ini

### Series dari List

```python
import pandas as pd

data = [1, 2, 3, 4, 5]
series_data = pd.Series(data)
print(series_data)
```

Hasil kode diatas maka akan seperti di bawah

```
0    1
1    2
2    3
3    4
4    5
dtype: int64
```

### Series dari Dictionary

```python
import pandas as pd

data = {"a": 1, "b": 2, "c": 3}
series_data = pd.Series(data)
print(series_data)
```

Hasil kode diatas maka akan seperti di bawah

```
a    1
b    2
c    3
dtype: int64
```

Dari kode di atas, apakah Anda sudah tahu perbedaan nya? Mari kita bahas.
Jika Series dari list ini untuk indeks nya akan secara otomatis dibuatkan dan jenis nya itu sebuah integer dari 0 sampai panjang data, kenapa otomatis? karena list ini tidak memiliki label sebagai indeks nya, maka dibuatkan secara otomatis dari angka. Lain hal dengan Series dari dictionary, pada dictionary itu memiliki `key` sebagai indeks nya, jadi `key` itu lah yang akan digunakan sebagai indeks.

## Apa itu DataFrame?

DataFrame ini sebuah struktur data tabular dua dimensi dengan baris dan kolom. Jadi yang dimaksud dengan tabular yaitu sebuah data yang disusun dalam bentuk tabel yang terdiri dari baris (horizontal) dan kolom (vertical). Jika kamu sudah tahu spreadsheet atau table pada database relational, maka DataFrame ini mirip dengan konsep tadi, yang dimana DataFrame ini terdiri dari baris dan kolom.

Mari kita contohkan, misal saya memiliki data populasi penduduk dari masing-masing provinsi yang ada di indonesia, saya ambil 5 sampel data populasi terbanyak.

| Nama Provinsi  | Populasi Penduduk (Ribu Jiwa) |
| -------------- | ----------------------------- |
| Jawa Barat     | 50.345,2                      |
| Jawa Timur     | 41.814,5                      |
| Jawa Tengah    | 37.892,3                      |
| Sumatera Utara | 15.588,5                      |

> [!NOTE]
> Data ini diambil berdasarkan data [Badan Pusat Statistik Provinsi Sulawesi Utara](https://sulut.bps.go.id/id/statistics-table/2/OTU4IzI=/jumlah-penduduk-menurut-provinsi-di-indonesia.html) yang terakhir diperbarui pada 20 September 2024.

Contoh kode untuk data di atas yaitu:

```python
import pandas as pd
data = {
    "provinsi": ["Jawa Barat", "Jawa Timur", "Jawa Tengah", "Sumatera Utara"],
    "populasi": [50345.2, 41814.5, 37892.3, 15588.5]
}
df = pd.DataFrame(data)
df.columns = ["Nama Provinsi", "Populasi Penduduk (Ribu Jiwa)"]
print(df)
```

Dari kode diatas maka akan menghasilkan output seperti di bawah ini:

```
    Nama Provinsi  Populasi Penduduk (Ribu Jiwa)
0      Jawa Barat                        50345.2
1      Jawa Timur                        41814.5
2     Jawa Tengah                        37892.3
3  Sumatera Utara                        15588.5
```

# Dasar Pandas

Pandas ini sebenarnya terlalu overpower, yang dimana selain bisa melakukan manipulasi data, Pandas juga bisa dipakai untuk melakukan analisa menggunakan perhitungan matematis. Mari kita mulai dari hal yang paling sederhana dahulu yaitu Manipulasi Data, kemudian kita akan melakukan Analisa Data. Untuk melakukan praktik ini, kita akan tetap menggunakan data yang sama dengan data Populasi Terbanyak di Indonesia.

## Manipulasi Data

Manipulasi data merupakan salah satu kemampuan inti dari Pandas, yang dimana kita dapat memilih, menambah, menghapus, mengurutkan sampai menyaring data dengan mudah. Mari kita coba...

### Memilih Kolom

```python
print(df["Nama Provinsi"])
```

Hasil dari kode diatas:

```
0        Jawa Barat
1        Jawa Timur
2       Jawa Tengah
3    Sumatera Utara
Name: Nama Provinsi, dtype: object
```

### Memilih Baris

Pandas memiliki dua cara untuk memilih baris, yaitu menggunakan `loc` dan `iloc`. Bedanya, jika `loc` ini mengambil baris berdasarkan label index, sedangkan jika menggunakan `iloc` ini akan mengambil baris berdasarkan posisi, sama seperti slicing pada list Python biasa. Mari kita coba...

```python
print(df.loc[0:2])
```

Hasil dari kode diatas:

```
  Nama Provinsi  Populasi Penduduk (Ribu Jiwa)
0    Jawa Barat                        50345.2
1    Jawa Timur                        41814.5
2   Jawa Tengah                        37892.3
```

Lalu jika menggunakan `iloc`

```python
print(df.iloc[0:2])
```

Hasil dari kode yang menggunakan `iloc`

```
  Nama Provinsi  Populasi Penduduk (Ribu Jiwa)
0    Jawa Barat                        50345.2
1    Jawa Timur                        41814.5
```

Nah coba perhatikan perbedaannya, jika menggunakan `loc[0:2]` itu mengambil baris dari index 0 sampai 2, sedangkan jika menggunakan `iloc[0:2]` mengambil baris posisi dari 0 sampai 2. Dari konsep ini, banyak pengguna yang merasa kebingungan bagi pemula, termasuk saya :)

### Menambah Kolom Baru

Menambah kolom baru pada DataFrame sebenarnya cukup mudah, kita perlu menyebutkan nama kolom baru lalu memberikannya nilai.

Contohnya yaitu:

```python
df["Pulau"] = ["Jawa", "Jawa", "Jawa", "Sumatera"]
print(df)
```

Hasil kode diatas maka akan seperti di bawah:

```
    Nama Provinsi  Populasi Penduduk (Ribu Jiwa)     Pulau
0      Jawa Barat                        50345.2      Jawa
1      Jawa Timur                        41814.5      Jawa
2     Jawa Tengah                        37892.3      Jawa
3  Sumatera Utara                        15588.5  Sumatera
```

### Menghapus Kolom atau Baris

Untuk menghapus kolom atau baris, kita bisa menggunakan method `drop()`. Secara default, `drop()` ini tidak mengubah DataFrame asli, melainkan mengembalikan DataFrame baru, kecuali kita menambahkan parameter `inplace=True`.

```python
df_baru = df.drop(columns=["Pulau"])
print(df_baru)
```

Hasil kode diatas maka akan seperti di bawah

```
    Nama Provinsi  Populasi Penduduk (Ribu Jiwa)
0      Jawa Barat                        50345.2
1      Jawa Timur                        41814.5
2     Jawa Tengah                        37892.3
3  Sumatera Utara                        15588.5
```

Begitu juga jika kita ingin menghapus baris, kita bisa menyebutkan indeks nya saja pada paramter `index`.

```python
print(df.drop(index=0))
```

Hasil kode diatas maka akan seperti di bawah

```
    Nama Provinsi  Populasi Penduduk (Ribu Jiwa)     Pulau
1      Jawa Timur                        41814.5      Jawa
2     Jawa Tengah                        37892.3      Jawa
3  Sumatera Utara                        15588.5  Sumatera
```

### Mengurutkan Data

Untuk mengurutkan data, gunakan method `sort_values()` dengan menyebutkan kolom mana yang ingin dijadikan acuan urutan.

```python
print(df.sort_values(by="Nama Provinsi"))
```

Hasil kode diatas maka akan seperti di bawah

```
Nama Provinsi  Populasi Penduduk (Ribu Jiwa)     Pulau
0      Jawa Barat                        50345.2      Jawa
2     Jawa Tengah                        37892.3      Jawa
1      Jawa Timur                        41814.5      Jawa
3  Sumatera Utara                        15588.5  Sumatera
```

Data diatas sekarang sudah berurutan berdasarkan abjad nama Provinsi nya.

### Menyaring Data (Filtering)

Pandas memungkinkan kita menyaring baris data berdasarkan kondisi tertentu.

```python
print(df[df["Populasi Penduduk (Ribu Jiwa)"] > 40000])
```

Hasil kode diatas maka akan seperti di bawah

```
Nama Provinsi  Populasi Penduduk (Ribu Jiwa) Pulau
0    Jawa Barat                        50345.2  Jawa
1    Jawa Timur                        41814.5  Jawa
```

Kode diatas hanya menampilkan provinsi dengan populasi lebih dari `40.000` ribu jiwa.

## Analisa Data

Selain melakukan manipulasi data, Pandas juga dapat melakukan analisa data secara cpeat, mulai dari statistik deskriptif sampai pengelompokan data.

### Statistik Deskriptif

Method `describe()` akan menampilkan ringkasan statistik dari sebuah kolom numerik, seperti rata-rata, nilai minimum, nilai maksimum, dan beberapa persentil.

```python
print(df["Populasi Penduduk (Ribu Jiwa)"].describe())
```

Hasil kode diatas maka akan seperti di bawah

```
count        4.000000
mean     36410.125000
std      14822.623416
min      15588.500000
25%      32316.350000
50%      39853.400000
75%      43947.175000
max      50345.200000
Name: Populasi Penduduk (Ribu Jiwa), dtype: float64
```

### Mengelompokkan Data

Method `groupby()` digunakan untuk mengelompokkan data berdasarkan kategori tertentu, lalu melakukan agregasi (penjumlahan, rata-rata, dll) pada masing-masing kelompok. Disini kita akan mengelompokkan data berdasarkan kolo `Pulau` yang sudah kita buat sebelumnya.

```python
print(df.groupby("Pulau")["Populasi Penduduk (Ribu Jiwa)"].sum())
```

Hasil kode diatas maka akan seperti di bawah

```
Pulau
Jawa        130052.0
Sumatera     15588.5
Name: Populasi Penduduk (Ribu Jiwa), dtype: float64
```

Dari hasil diatas terlihat jika total populasi dari sampel provinsi di Pulau Jawa jauh lebih besar dibandingkan Pulau Sumatera.

## Peningkatan Terbesar (2010 sampai 2024)

    df_tahun_2010[['kode_provinsi', 'nama_provinsi', 'rata_rata_lama_sekolah']],
